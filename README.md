# <omoide>
## サイト概要
みんなの旅の思い出写真を共有できるSNSサイト
### サイトテーマ
旅の思い出の写真を地図でまとめてみませんか？

### テーマを選んだ理由
地図付きで思い出写真を紹介できたら、どこで撮影したか見てわかりやすく、
旅行の思い出ももっと蘇るのではと思いこのテーマにしました。

### ターゲットユーザ
旅好きな方

### 主な利用シーン
旅行後、旅行中、あなたや仲間が撮影したこの一枚！というものを紹介してくだい！
